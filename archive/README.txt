These are the old "run from source" launchers for AutoCut.

They are kept for reference only and are NOT the recommended way to use AutoCut.
They require you to install Python and ffmpeg yourself, which most people should
not need to do.

Instead, download the ready-to-run installer:
  https://tellinghouse-autocut.netlify.app
  (or from https://github.com/tellinghouse/autocut/releases/latest)

The installer bundles everything -- no Python, no ffmpeg, nothing else to set up.
